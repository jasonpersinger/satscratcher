## Blank Screen

Subject: Re: SatScratcher screen is blank

Try a different USB cable first. A lot of USB cables only carry power poorly or are just tired.

If that does not work, send me:

- Order number
- Photo of the screen while plugged in
- What USB power source you tried

## WiFi Setup

Subject: Re: SatScratcher WiFi setup

The SatScratcher network only appears during setup mode. If the device already joined your WiFi once, it will not keep broadcasting its own network.

Unplug it, plug it back in, and watch the first 30 seconds. If the SatScratcher network never appears, send me a photo of the screen.
