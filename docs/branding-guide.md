# SatScratcher Brand Guide

SatScratcher is pixel-art retro-gaming kawaii: cute mascot, dry copy, honest math.

## Promise

Mine Bitcoin. Win (Maybe). Look Cool Regardless.

## Palette

| Token | Hex | Role |
|---|---|---|
| `--bg` | `#252627` | Page background |
| `--bg-elevated` | `#2E3032` | Elevated surfaces |
| `--text` | `#CDD3D5` | Body text |
| `--primary` | `#FB6107` | CTAs, logo, active links |
| `--jackpot` | `#F5A623` | Prices and win moments |
| `--signal` | `#1B998B` | Mining-active indicators |
| `--danger` | `#C2402C` | Errors |

Use `--primary` for action and `--jackpot` for celebration. Do not put both on the same element.

## Type

- Display: Press Start 2P
- Body: Inter

Use the display font sparingly. It is for short headlines, counters, and pixel UI moments.

## Voice

- Honest about the math.
- Self-deprecating, but confident.
- Short and dry.
- Cute visuals, deadpan copy.
- No Bitcoin-maxi slang.

## Do Not

- No laser eyes.
- No HODL jokes.
- No Shiba/Doge references.
- No fake winner testimonials.
- No claims that imply likely profit.
